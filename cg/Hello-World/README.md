# Hello-World

- Install 
    ```
    sudo apt-get install build-essential
    sudo apt install xorg-dev libglu1-mesa-dev freeglut3-dev mesa-common-dev
    ```

- Install `cmake`
    ```
    sudo apt install cmake
    ```

- Run following commands:

    ```
    mkdir build
    cd build
    cmake ..
    make
    ./Hello-World
    ```
